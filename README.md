# Topological-Finance
A repository going more in depth about applications of Topology as they relate to Financial Markets. 

## Problem
We want to find the most profitable strategy to trade the market given a list of stocks. Given the project name we will use Topological Data Analysis to do so.

## Methodology
We will first start by examining bounded metrics and how they are useful in stabilizing financial market data. Then we will make use of persistent homologies and simplicial complexes to find holes in order to identify market crashes.

## Why does this matter?
Topological Data Analysis is a relatively new field of mathematics with promising applications in finance, and other fields where high dimensional, noisy data is ubiquitous. By exploring how to best trade with it, we could open up doors to potentially better trading strategies.
